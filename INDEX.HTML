Fundamentos de HTML y CSS – Parte Teórica

1. Formas de seleccionar elementos en CSS

Selector por etiqueta: Selecciona todas las etiquetas del mismo tipo. Ejemplo: p { color: red; } cambia el color a todos los párrafos.

Selector por clase: Selecciona elementos que tienen un atributo class. Se usa un punto. Ejemplo: .caja { background: blue; }.

Selector por ID: Se usa para identificar un elemento único. Se escribe con # Ejemplo: #titulo { font-size: 30px; }.

Selectores universales y agrupados: El selector universal * aplica estilo a todos los elementos. Los selectores agrupados permiten aplicar el mismo estilo a varios elementos: Ejemplo: h1, h2, p { margin: 10px; }.

2. Modelos de color en CSS

Color por nombre: Colores predefinidos como red, blue, green.

RGB: Mezcla de colores usando valores entre 0 y 255. Ejemplo: rgb(255, 0, 0).

Hexadecimal: Representa los colores con valores hexadecimales. Ejemplo: #FF0000.

3. Propiedad Display

display: block → Ocupa todo el ancho disponible. Ej: div, h1, p.

display: inline → Ocupa solo el espacio necesario. Ej: span, a.

display: inline-block → Permite tamaño propio pero se ubica en línea.

4. Position Relative y Absolute

position: relative: Sirve para mover un elemento desde su posición original.

position: absolute: Se posiciona respecto al elemento padre que tenga position: relative.

Ambos se usan juntos para colocar elementos flotantes o decorativos.
